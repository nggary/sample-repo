# Sample iOS App

A sample iOS application built with Swift and SwiftUI.

## Features

- Modern SwiftUI interface
- Simple counter functionality
- Clean architecture
- Swift 5.9+
- iOS 17+ support

## Requirements

- Xcode 15.0+
- iOS 17.0+
- Swift 5.9+

## Getting Started

1. Clone the repository
2. Open the project in Xcode
3. Build and run the app on simulator or device

## Project Structure

```
SampleApp/
├── SampleApp.swift       # App entry point
├── ContentView.swift     # Main view
└── Assets.xcassets/      # Asset catalog
```

## Swift Package Manager

This project uses Swift Package Manager for dependency management. You can add dependencies in the `Package.swift` file.
